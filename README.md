## 使用说明

- [下载发布包](https://github.com/DaiMao204/MaaResonance/releases) - 下载 Windows 测试包并运行 `MaaResonance.exe`。
- [新手上路](#使用说明) - 当前仅支持安卓模拟器 ADB 连接，不支持 PC 端窗口控制。
- [功能介绍](#功能列表) - 查看当前首版支持的功能范围和测试状态。
- [连接设置](#使用说明) - 建议使用 16:9 横屏模拟器，短边缩放为 720。
- [常见问题](./docs/zh_cn/develop/faq.md) - 遇到问题可先查看这里。
- [MaaPiCli 使用说明](https://github.com/MaaXYZ/MaaFramework) - 命令行版使用指南。
