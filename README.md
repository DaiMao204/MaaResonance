## 使用须知

- 推荐使用 MuMu 模拟器。
- 推荐分辨率：16:9，`1920x1080` 或 `1280x720`。
- 当前仅支持安卓模拟器 ADB 连接，不支持 PC 端窗口控制。
- 下载发布包后运行 `MaaResonance.exe`。
